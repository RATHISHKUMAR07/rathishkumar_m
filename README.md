## 🛠️ Tech Stack & Architecture

This application is built entirely as a high-performance, serverless frontend client that reads dynamic structured data from central configurations.

- **Frontend Core:** React.js (Functional Components, Hooks)
- **Styling Paradigm:** CSS Modules (for isolated component scoping)
- **Observability Stack:** Styled matching enterprise ELK (Elasticsearch, Logstash, Kibana) dashboards
- **Forms Layer:** Serverless integration via EmailJS SDK (Zero-Backend secure mail delivery)
- **Build Tooling:** Vite / Create React App

## 🚀 Local Development Setup

To clone the repository and run the development server locally on your machine, follow these steps:

```bash
# 1. Clone the repository
git clone [https://github.com/rathishkumar07/your-portfolio-repo.git](https://github.com/rathishkumar07/your-portfolio-repo.git)

# 2. Navigate into the project root directory
cd your-portfolio-repo

# 3. Install the project dependencies (including @emailjs/browser)
npm install

# 4. Start the local development server
npm run dev  # or 'npm start' if using Create React App
